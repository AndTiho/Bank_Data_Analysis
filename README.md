# Проект "Bank AD" 
## Описание
Учебный проект Bank Analys Data создан при поддержке SkyPro и является beck-end приложением для анализа
данных из банковских отчётов (в частности Т-банка) в формате excel.\
**В данный момент находится в разработке, просим проявить терпение.**
## Установка:
1. Клонируйте репозиторий:
```
git@github.com:AndTiho/Bank_Data_Analysis.git
```
2. Установите зависимости:
```
poetry install
```
## Использование
1. В модуле [Утилиты](src\utils.py) находятся:
- Функции отражающие общий функционал нашего приложения.
2. В модуле [Отчёты](src\reports.py) находятся:
- Функция, которая принимает данные транзакций, категорию на выбор пользователя и дату
(по умолчанию берётся сегодняшняя) и возвращает траты по заданной категории за последние три месяца.
3. В модуле [Сервисы](src\services.py) находятся:
- Функция, которая принимает на вход три аргумента: данные с транзакциями, год и месяц. 
На выходе мы получаем JSON-файл сколько на каждой категории заработано кэшбэка за выбранный месяц года
4. В модуле [Главная страница](src\views.py) находятся:
Функция, которая принимает на вход дату в формате YYYY-MM-DD HH:MM:SS и возвращает
JSON-ответ со следующими данными:
   * Приветствие;
   * По каждой карте:
     - последние 4 цифры карты;
     - общая сумма расходов;
     - кешбэк (1 рубль на каждые 100 рублей);
   * Топ-5 транзакций по сумме платежа;
   * Курс валют. Стоимость акций из S&P500.
5. В модуле [ГЛАВНАЯ](src\main.py) находится:
- Находится функция `main` которая запускает основной функционал программы.

## Запуск и ожидаемое поведение программы
Для запуска программы воспользуйтесь:
```
python src\main.py
```
Программа находится на стадии альфа-теста. Для работы программы прилагаются данные транзакций в формате xlsx,
которые можно использовать в качестве тестирования работы функции. Находятся они по адресу [Данные транзакций](data).

**Ожидаемое поведение программы должно быть следующим:**

Программа приветствует пользователя и предлагает на выбор отработать одну из предоставленных опций:\
"1. Данные для главной страницы."\
"2. Данные по выгодным категориям повышенного Кэшбэка."\
"3. Данные трат по выбранной категории."\
"4. Выход из программы."

Пользователь: 1\
Программа: Предлагает ввести дату в формате YYYY-MM-DD HH:MM:SS а так же вариант для копирования
и подстановки в программу, чтобы не гадать. 

Пользователь: 2020-10-02 16:05:00\
Программа: Генерирует ответ в формате JSON с данными:
```
            "greeting": Приветствие,
            "cards": [Данные по каждой карте],
            "top_transactions": Топ-5 транзакций по сумме платежа,
            "currency_rates": Курсы валют,
            "stock_prices": Курсы Акций
```
Примечание: наименования валют и акций берутся из файла [настройки](user_settings.json).\
Для курса валют используется сервис https://apilayer.com \
Для курса акций используется сервис https://financialmodelingprep.com

Пользователь: 2\
Программа: Предлагает ввести дату в формате YYYY-MM а так же вариант для копирования
и подстановки в программу, чтобы не гадать.

Пользователь: 2020-10\
Программа: Генерирует ответ в формате JSON с данными:
```
    "Аптеки": 57.0,
    "Транспорт": 8.0
```

Пользователь: 3\
Программа: предлагает список имеющихся категорий и просит ввести категорию и дату в формате YYYY-MM-DD а так же вариант для копирования
и подстановки в программу, чтобы не гадать.

Пользователь: Аптеки\
Пользователь: 2020-10-20\
Программа: генерирует в ответ JSON данные, а так же записывает их в отдельный файл:\
[Результат](data/results.txt)


Пользователь: 4\
Программы: Завершает свою работу.
